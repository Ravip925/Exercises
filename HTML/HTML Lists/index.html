HTML LISTS

Nested Order List:

Example:-

  1. Type 1
    1. Item a
    2. item b

  2. Type 2
    1. Item a
    2. Item b

  3. Type 3
    1. Item a
    2. Item b

Reverse Order List:

Example:-

  1. Type 1
    1. Item a
    2. item b

  2. Type 2
    1. Item a
    2. Item b

  3. Type 3
    1. Item a
    2. Item b

Unordered List:-

Example:-

  • Type 1
    • Item a
    • item b

  • Type 2
    • Item a
    • Item b

  • Type 3
    • Item a
    • Item b
